树的基本概念

  1. 树是有 n(n>=0)个节点的有限集合，n=0 时为 空树 ，在一颗非空的树中有且仅有一个根节点，当 n > 1 时，其余节点可以分为 m (m > 0) 个互不相交的有限集合，其中每个集合又是一颗树，并称为根的子树。

  2. 节点的子树的根称为该节点的孩子，该节点称为其孩子的双亲，同一节点的孩子之间互称为兄弟。

  3. 节点的祖先是从根节点到该节点途径的所有节点。

  4. 节点拥有的子树数目称为节点的度，度为 0 的节点称为叶子节点(终端节点) ，度不为0的节点称为分支节点(非终端节点) ，除根节点外，非终端节点又称为内部节点，树的度是各节点度的最大值。

  5. 以某节点为根的子树中任一节点都是该节点的子孙。

  1. 节点的层次是从根节点开始算起，根为第一层，其孩子为第二层，依次类推。树中最大层树即为树的深度。

  2. 双亲在同一层的节点互称为堂兄弟。

  3. 如果子树从左到右是有次序的，则称此树为有序树，反之为无序树。

二叉树

二叉树是 n(n>=0) 个节点的有限集合，n=0 时称为空二叉树，由一个根节点和两根互不相交，分别称为根节点左子树、右子树的两颗二叉树组成。

特点

  • 每个节点_最多_两颗子树，不存在大于 2 个度的节点。
  • 子树具有顺序，分为左子树和右子树。

分类

二叉树根据形态可以分为以下不同树：

  • 斜树

    所有节点都在同一边，比如左斜树所有节点只有左子树，树退化为链表这类线性结构。

  • 满二叉树

    所有节点都有左右子树（除了叶子节点），且所有叶子节点在同一层。

    其叶子节点只在最下一层，非叶子节点度一定为 2 ，同样深度的一颗二叉树，满二叉树节点最多，叶子节点最多。

  • 完全二叉树

    一颗具有 n 个节点的二叉树，从根节点开始编号，编号为 i (1 <= i <= n)的节点位置与节点数相同的满二叉树中编号为 i 的节点一样，则称为完全二叉树。

    特点：

    1. 叶子节点只能在最下两层。
    2. 最下层叶子一定集中在左部连续位置。倒数第二层叶子节点一定集中在右部连续位置。
    3. 若节点度数为 1 ，一定只有左孩子。
    4. 同样节点数的二叉树，完全二叉树深度最小。
    5. 具有 n 个节点的完全二叉树，其深度为 $log_{2}^{n} + 1$ 。

二叉树的性质

  1. 二叉树第 i(i>=0) 层上，至多有 $2^{i-1}$ 个节点。
  2. 深度为 k (k>=1) 的二叉树，至多有 $2^k - 1$ 个节点。
  3. 在有 n 个节点的二叉树中，若叶子节点数为 $n_0$ ，度为 2 的节点数为 $n_2$ ，则 $n = n_0 + n_2$ 。

void front_visit(TreeNode* node)
{
    if (nullptr == node) {
        return;
    }
    // 访问节点
    visit(node);
    // 前序遍历左节点
    fron_visit(node->left);
    // 前序遍历右节点
    front_visit(node->right);
}
// 非递归方式，用用户栈模拟函数栈
void front_visit(TreeNode* node)
{
    if (nullptr == node) {
        return;
    }
    stack<TreeNode*> s;
    // 根节点入栈
    s.push(node);
    while (!s.empty()) {
        auto n = s.top(); s.pop();
        // 前序遍历，先访问节点
        visit(n);
        // 右子树后访问，先入栈
        if (n->right) { s.push(n->right); }
        // 左子树先访问，后入栈
        if (n->left) { s.push(n->left); }
    }
}

void middle_visit(TreeNode* node)
{
    if (nullptr == node) {
        return;
    }
    middle_visit(node->left);
    visit(node);
    middle_visit(node->right);
}

// 非递归方式
void middle_visit(TreeNode* node)
{
    if (nullptr == node) {
        return;
    }
    stack<TreeNode*> s;
    auto n = node;
    while (!s.empty() || n) {
        // 节点不为空，如果左子树遍历完成，节点也是空，则一直入栈其左孩子
        while (nullptr != n) {
            s.push(n);
            n = n->left;
        }
        // 此时左子树入栈完成，开始访问
        if (!s.empty()) {
            n = s.top();
            s.pop();
            visit(n);
            // 遍历右子树，如果为空，上面循环不会再入栈
            n = n->right;
        }
    }
}

void back_visit(TreeNode* node)
{
    if (nullptr == node) {
        return;
    }
    back_visit(node->left);
    back_visit(node->right);
    visit(node);
}

// 非递归方式
void back_visit(TreeNode* node)
{
    if (nullptr == node) {
        return;
    }
    stack<TreeNode*> s;
    TreeNode* n = node, pre;
    bool flag;
    while (!s.empty() || n) {
        while (nullptr != n) {
            s.push(n); n = n->left;
        }
        pre = nullptr, flag = true;
        while (!s.empty() && flag) {
            n = s.top();
            if (n->right == pre) {
                s.pop(); visit(n); pre = n;
            }else {
                n = n->right; flag = false;
            }
        }
    }
}

    // 从第一层开始逐层打印，打印二叉树第K层，相当于以根节点左孩子为根节点，打印第 k - 1 层，
// 然后以根节点右孩子为根节点，打印第 k - 1 层。
// 递归实现
void _level_visit(TreeNode* node, int level)
{
    if (nullptr == node || level < 1) {
        return;
    }
    if (level == 1) {
        visit(node);
    }
    _level_visit(node->left, level - 1);
    _level_visit(node->right, level - 1);
}
// 计算二叉树层数
int getTreeDeepth(TreeNode* node) {
    if (nullptr == node) {
        return 0;
    }
    return std::max(getTreeDeepth(node->left), getTreeDeepth(node->right) + 1);
}
void level_visit(TreeNode* node)
{
    if (nullptr == node) {
        return;
    }
    // 计算二叉树层数
    int deepth = getTreeDeepth(node);
    for (int i = 1; i <= deepth; ++i) {
        _level_visit(node, i);
    }
}

// 非递归使用队列存储节点，先让根节点入队，然后访问节点并判断节点是否有左右孩子，
// 有就入队，直到队列为空。
// 非递归实现
void level_visit(TreeNode* node)
{
    if (nullptr == node) {
    	return;
   	}
    deque<TreeNode*> q;
    q.push_back(node);
    while (!q.empty()) {
        auto n = q.front();
        q.pop_front();
        visit(n);
        if (n->left) { q.push_back(n->left); }
        if (n->right) { q.push_back(n->right); }
    }
}
